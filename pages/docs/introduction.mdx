Welcome to Bauto Framework! This guide will help you understand how to use Bauto Framework, from installation to advanced usage. It is intended to be read by those who wish to learn more about the framework and how it works, as well as those who wish to build a project with Bauto Framework. It assumes you have a basic knowledge of programming, but tries to be as simple and easy to understand as possible so that everyone can learn and benefit.

## Installation

Bauto Framework is available on [Packagist](https://packagist.org/packages/ricomuh/bauto) (using semantic versioning), and installation via [Composer](https://getcomposer.org) is the recommended way to install Bauto Framework. Just use the following command to create a new project from scratch:

```bash
composer create-project ricomuh/bauto "0.1.2-alpha"
```

## Configuration

After creating a new project, you should copy the `.env.example` file to `.env` and configure your application's database connection details - then you'll be ready to execute the `migrate` command.
